## 🚀 Getting Started

### Immediate (5 minutes)
```bash
# 1. Install
pnpm install

# 2. Setup env
cp .env.example .env.local

# 3. Run
npm run dev

# 4. Visit
http://localhost:3000
```
All features work with mock data immediately.

### With Real Data (15 minutes)
1. Create Supabase account (free)
2. Run SQL scripts in Supabase dashboard
3. Copy 3 API keys to `.env.local`
4. Refresh browser
5. Done!

### Deploy to Vercel (5 minutes)
```bash
# Push to GitHub (or deploy directly)
git push origin main
```
Vercel auto-detects and deploys.
